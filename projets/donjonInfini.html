Donjon Infini
Projet de BUT Informatique, en java.
Attendus
L’objectif de ce projet de but Informatique est d’implémenter un jeu simple d’exploration de donjon.
Le projet se divise en deux partie :
  • Dans la première partie, nous avons codé le jeu de base avec les spécificités fournies par le sujet, Il y avait notamment un Diagramme de cas d’usage centré sur le contrôleur, un diagramme de classes de l’application ainsi qu’un diagramme séquence illustrant un tour de jeu.
  • Dans un deuxième temps, nous avonas été amené.e.s à continuer le développement du jeu, en implémen- tant de nouvelles fonctionnalités de notre choix.

Nouvelles fonctionnalités implémentée :
  • Différents types de monstres.
  • Différents types d'armes.
  • Un nouvel item.
  • Sprites animés.
  • Musique.
  • Menu.
  • Paramètres
